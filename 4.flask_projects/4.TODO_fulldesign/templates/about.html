Flask 예제 애플리케이션 소개

이 애플리케이션에 대해

이 Flask 예제 애플리케이션은 웹 개발을 배우는 분들을 위한 간단한 할 일 관리 시스템입니다. Flask 프레임워크의 기본적인 기능들을 실습할 수 있도록 구성되었습니다.

주요 기능
  • ✅ 할 일 추가, 수정, 삭제
  • ✅ 할 일 완료 상태 관리
  • ✅ RESTful API 제공
  • ✅ 반응형 웹 디자인
  • ✅ 실시간 시간 표시
사용된 기술
백엔드
  • Python 3.x
  • Flask 웹 프레임워크
  • Jinja2 템플릿 엔진
프론트엔드
  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
API 엔드포인트
메서드 엔드포인트 설명
GET /api/todos 모든 할 일 목록 조회
POST /api/todos 새로운 할 일 추가
PUT /api/todos/<id> 특정 할 일 수정
DELETE /api/todos/<id> 특정 할 일 삭제
GET /api/time 현재 시간 조회
실행 방법

이 애플리케이션을 실행하려면 다음 단계를 따르세요:

  1. Python 3.x가 설치되어 있는지 확인
  2. Flask 설치: pip install flask
  3. 애플리케이션 실행: python app.py
  4. 브라우저에서 http://localhost:5000 접속
